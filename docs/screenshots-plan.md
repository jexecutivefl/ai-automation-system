## Capture Instructions

1. Run `npm run dev`
2. Navigate to `/demo` → click "Seed Demo Data"
3. Set browser window to 1440x900
4. Capture each screenshot per the states above
5. Use browser DevTools screenshot (Cmd+Shift+4 on Mac) for clean captures
6. Save as PNG, naming: `screenshot-01-dashboard.png`, `screenshot-02-requests.png`, etc.
